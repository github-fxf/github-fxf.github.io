# Java开发笔记之常用判空操作

#  List判空方法：

1. CollectionUtils.isEmpty(list) --判断list是否为空
2. CollectionUtils.isNotEmpty(list)  --判断list是否不为空

# Map判空方法：

1. MapUtils.isEmpty(map) --判断map是否为空

2. MapUtils.isNotEmpty(map)  --判断map是否不为空

# String判空方法

StringUtils.isBlank (string) --判断string是否为空
